# FikStream - Premium Streaming Platform

FikStream is a modern, high-performance streaming application built with React and Vite. It features a sleek, cinematic dark mode interface designed for the best viewing experience.

## Features

- **Cinematic Dark UI**: Immersive viewing experience with "Midnight Slate" theme.
- **Multi-Category Support**: Movies, K-Drama, C-Drama, and Anime.
- **Fast Performance**: Powered by React + Vite for instant loading.
- **Responsive Design**: Optimized for Desktop, Tablet, and Mobile.
- **Smart Search**: Find your favorite content instantly.

## Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Vanilla CSS (Custom Variable System), Lucide React Icons
- **Routing**: React Router DOM

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Development Server**
    ```bash
    npm run dev
    ```

3.  **Build for Production**
    ```bash
    npm run build
    ```
